# 🧪 Relatório Diário de Laboratório

## 📖 Sobre o Projeto

Este projeto foi desenvolvido durante a graduação em Análise e Desenvolvimento de Sistemas com o objetivo de aplicar conceitos fundamentais de programação em Java.

A aplicação recebe informações sobre amostras processadas em um laboratório e calcula indicadores importantes para análise de desempenho operacional, como o tempo médio por amostra e o percentual de resultados válidos.

Além da resolução do problema proposto, o projeto foi utilizado como exercício para praticar:

* Estruturação de programas em Java
* Criação de métodos reutilizáveis
* Separação de responsabilidades
* Operações matemáticas
* Entrada e saída de dados
* Formatação de resultados
* Boas práticas para evitar repetição de código (DRY - Don't Repeat Yourself)

---

## 🚀 Funcionalidades

* Registro da quantidade de amostras processadas
* Registro do tempo total gasto
* Registro da quantidade de resultados válidos
* Cálculo do tempo médio por amostra
* Cálculo do percentual de resultados válidos
* Exibição dos resultados formatados com duas casas decimais

---

## 🛠️ Tecnologias Utilizadas

* Java
* Scanner (entrada de dados)
* Locale
* Métodos estáticos
* Operações matemáticas

---

## 📊 Fórmulas Aplicadas

### Tempo Médio por Amostra

tempo_medio = tempo_total / quantidade_de_amostras

### Percentual de Resultados Válidos

percentual_validos = (resultados_validos / quantidade_de_amostras) * 100

## 💻 Exemplo de Execução

```text
======= Relatório Diário do Laboratório =======

Quantidade de Amostras Processadas: 50

Número de Resultados Válidos: 45
```
